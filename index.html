QuickWand

Fast draw but with TinyML gestures
GitHub

Setting up the Hardware

Upload magic_wand.ino on two Arduino Nano 33 BLEs using the Arduino IDE. It determines the gestures using a TensorFlow Lite Micro model and sends the data over Bluetooth. Attach the Arduinos—with a micro USB cable plugged in—to sticks using tape.

Connecting using Bluetooth

Make sure that your browser supports Web Bluetooth and that it is enabled. Once both of the Arduinos are on, use the Connect Player One and Connect Player Two buttons. Clicking on the buttons will result in a pop-up that allows you to connect to the Arduinos. You can tell which Arduino connects to which player as the orange light on the Arduinos turns on when connected. Make sure you connect to a different device for each player, and you can tell which device represents which player as the orange light on the Arduinos turns on it is connected.

Playing the Game

Once you connect both Arduinos, you can start the game. In the first and second rounds, whoever draws the gesture in the air first gets the point. In the third round, you compete to draw three in a row. Whoever has the most points at the end wins.
